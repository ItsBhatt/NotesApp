# Notes-APP

> Simple Notes App
> A MEVN project (MongoDB, Express.js, Vue, Node.js)

### Project Setup

```bash
# Create .env file
> touch .env

#add MONGODB_URI="YOUR_DB_CREDENTIALS" in .env file

```

## Build Setup

```bash

# Setup & Run server

npm install
npm run serve


# Run client
cd client/
npm install
npm run dev


```
